# RSP - Semestrálka - Muzikál

Hosting: [App](https://rsp-web-muzikal.web.app/)

## Instalace

1. Nainstalujte si [Node.js](https://nodejs.org/en/)
2. V source složce spusťte příkaz:
```
npm i
```

## Důležité příkazy

Pro spuštění:
```
npm start
```

Pro deploy hosting:
```
npm run build
firebase deploy
```
